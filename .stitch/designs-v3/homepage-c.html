When agents share memory,
they share mind.

Telepathine is the cross-agent coordination layer. Deploy shared memory graphs, real-time signal routing, and hybrid retrieval for swarms of autonomous agents.

$ npm install @telepathine/sdk
telepathine.ts 
import { Telepathine } from "@telepathine/sdk";

const core = new Telepathine({
  apiKey: process.env.TLP_KEY,
  namespace: "research-swarm"
});

// Initialize shared memory bridge
await core.memory.sync({
  persistence: "graph",
  depth: 2
});

// Broadcast coordination signal
core.relay.broadcast("TASK_COMPLETE", {
  agentId: "worker-01",
  payload: { path: "s3://results/04" }
});
