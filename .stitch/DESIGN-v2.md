# Telepathine.ai — Design System

## Brand
Telepathine.ai — The channel layer for AI agents. When agents share memory, they share mind.

## Visual Identity
Developer API platform. Code terminals, wireframe diagrams, data tables, network topology. Think Linear.app, Vercel.com, Supabase.com — clean dark UI with warm amber accents.

## Color Palette

### Primary
- Amber Resin: #D4913D — primary accent, CTAs, active states, links
- Bark Black: #110F0D — deepest background
- Forest Soil: #161311 — page background
- Warm Panel: #1E1A16 — card/panel surfaces
- Moss Card: #1A1613 — elevated card surfaces

### Text
- Warm Parchment: #E8DCC8 — primary text, headings
- Copy Parchment: #E0D4BE — body copy
- Muted Bark: #A8A090 — secondary text, labels, captions
- Cedar Hover: #221E18 — hover state backgrounds

### Accent
- Vine Green: #5A8A5E — success states, checkmarks, status indicators
- Moss Light: #7AAB7E — lighter success accent

### Borders
- Outline Variant: #514437 — dividers, card borders
- Surface Variant: #373432 — subtle borders

## Typography

### Font Stack
- Headlines: Instrument Serif (italic for taglines) — literary warmth, sparing use
- Body: Geist Sans — clean, technical, readable
- Code/Labels: JetBrains Mono — terminal aesthetic, API references, install commands

### Scale
- H1 (hero): 3rem/1.1, Instrument Serif, weight 400
- H2 (section): 2rem/1.2, Instrument Serif, weight 400
- H3 (feature): 1.25rem/1.4, Geist Sans, weight 600
- Body: 1rem/1.7, Geist Sans, weight 400
- Body LG: 1.125rem/1.7, Geist Sans, weight 400
- Label: 0.875rem/1.5, JetBrains Mono, weight 500, tracking 0.05em
- Code: 0.875rem/1.5, JetBrains Mono, weight 400

## Layout

### Spacing
- Container max: 1280px
- Desktop margin: 4rem
- Mobile margin: 2rem
- Gutter: 1.5rem
- Section gap: 6rem
- Component gap: 2rem

### Structure
- Asymmetric, left-aligned layouts — NOT centered
- Hero: 7/5 column split (text left, code terminal right)
- Features: zig-zag alternating (text-image-text-image)
- Pricing: 3-column grid, center card elevated
- Footer: minimal, dark

## Components

### Navigation
- Sticky header, warm-panel background, backdrop-blur
- Logo (hex mark + text), nav links (Features, API, Docs, Pricing, Security), CTA button

### Hero Section
- Left: H1 serif headline, subtitle sans, install command line, two CTA buttons
- Right: Code editor/terminal window showing API usage — dark code block with syntax highlights

### Feature Cards
- Icon (Material Symbols), H3 title, description paragraph
- Visual: code terminal or wireframe diagram, NOT illustration or photography
- Alternating left/right layout with dividers

### Code Blocks
- Dark background (#110F0D), rounded corners, subtle border
- Syntax: amber for keywords, vine green for strings, parchment for variables
- Copy button in top-right corner

### Pricing Cards
- 3-column: Free / Pro ($29/mo) / Enterprise (custom)
- Pro card: amber border, elevated, "Popular" badge
- Feature lists with vine green checkmarks
- CTA buttons: amber primary, ghost secondary

### Buttons
- Primary: amber background, parchment text, subtle shadow
- Ghost: transparent, amber border, amber text
- Hover: lighten background, subtle lift

## Visual Elements
- Code terminals with syntax-highlighted API calls
- Wireframe network topology diagrams (geometric hex nodes, straight-line connections)
- Data tables with real API response payloads
- Status indicators with vine green dots
- Hexagonal grid pattern as subtle background texture (geometric, mesh-like)

## What This Is NOT
- NOT a nature/organic/botanical website
- NOT illustrations of mushrooms, mycelium, forests, bark, or plants
- NOT gradient orbs, glowing blobs, or twilight lighting
- NOT centered hero with floating illustrations
- NOT biological terminology (spores, hives, swarms, vines as nav items)
