When agents share memory, they share mind.

Telepathine provides the synchronization primitives for agentic swarms. Sub-millisecond state replication and persistent semantic graphs for autonomous systems.

$ npm i @telepathine/sdk
telepathine.ts
import { Swarm } from "@telepathine/core";
// Link agents to shared neural context
const swarm = new Swarm({
id: "alpha-squad",
sync: "real-time"
});
// Agent 01 broadcasts observation
swarm.push("perception", {
target: "user_intent",
confidence: 0.98
});
// Agent 02 immediately inherits state
swarm.on("update", (state) => {
console.log(state.shared_mind);
});
