**The market bet:** Every AI vendor wants to own your memory. Claude's memory only works with Claude. ChatGPT's only with ChatGPT. Telepathine is the only engine-agnostic, agent-centric memory layer — the Switzerland of AI agent memory.

## 2. What We Learned from the Research

### Research Insight 1: The 90/10 Balance Is Non-Negotiable
The brand's soul lives in the tension between hard engineering precision (90%) and subtle organic/mythological depth (10%). Go above 10% and you enter psychedelic cliché territory that kills developer trust. Go below and you become indistinguishable from Mem0 or Zep. The 10% is *texture*, never *subject matter*.

### Research Insight 2: "Engine-Agnostic" Is Unclaimed Territory
Mem0 is SDK-coupled. Zep killed self-hosting. Letta's memory is locked to its framework. Anthropic is protocol-first but has no standalone memory product. OpenAI has zero API for its 700M-user memory. The quadrant is genuinely empty — but the window is 12–18 months.

### Research Insight 3: The Bus Is the Real Differentiator
Memory alone is becoming table stakes. The real-time agent communication bus — with zero competitors shipping it — is Telepathine's moat. Every messaging system should lead with the bus, then explain memory as the substrate beneath it.

### Research Insight 4: LEAK-01 Is a Trust Asset, Not a Liability
We permanently removed a default project fallback, backfilled data, and wrote regression tests. Most companies hide incidents. By documenting it publicly, we create an unforgeable trust signal. This is the strongest security proof point we have.

### Research Insight 5: "Telepathine" Is Legally Risky but Salvageable
The trademark research reveals severe risks: Neuralink/Prophetic "TELEPATHY" marks, Telepath.io in the same vertical, USPTO deceptively misdescriptive risk for a known chemical compound, and payment processor flags. The name is *not* fatal if we:
- Lead 100% with etymology (tele + pathine = feeling at a distance)
- Never use biological/chemical metaphors in public copy
- Register `telepathine.io` defensively
- Accept that legal defense may be needed (budget ~$15-25K)

### Research Insight 6: SEO/GEO Is a Category-Creation Game
Nobody searches for "telepathine." They search for "AI agent memory" and "MCP memory server." The brand maps to search through category-defining pages, not brand-name stuffing. We must *own* the category terms before competitors do.

### Research Insight 7: Visual DNA Is Already Defined
The visual system (teal + amber on charcoal, serif headlines, hex grid, event-driven motion) creates a unique quadrant no competitor occupies. The hard work is done — implementation is now execution, not exploration.

## 9. Trust / Security Narrative

### The Core Principle
"Telepathine" carries psychedelic/esoteric connotations. In AI infrastructure, this creates a trust deficit. The fix: make security so concrete, specific, and un-marketing-fluffy that the name becomes a strength.

### Security Page Tone = Engineering Postmortem
Dry, specific, evidenced. The features page can use dream compression metaphors. Security must read like architecture. This contrast proves range.

### LEAK-01 Strategy: Lead With the Incident
> "We learned data isolation the hard way." In May 2026, a default project fallback caused 6,500+ observations from 5 projects to share a single namespace. We permanently removed the fallback, backfilled all data, and wrote regression tests with a standing instruction: *"If this test fails, DO NOT change the test. Fix the installer code."*

**Honest compliance roadmap:**
| Standard | Status | What to Say |
|----------|--------|-------------|
| RLS (FORCE policies) | ✅ Live | "Every table, FORCE policies" |
| BYOK | ✅ Live | "Your keys, your models" |
| Export | ✅ Live | "Full REST API, no lock-in" |
| SOC2 Type I | 🔄 Q3 2026 | "In progress" |
| SOC2 Type II | 🔄 2027 | "Targeting Q3 2027" |
| GDPR | 🔄 Partial | List what we have, don't claim compliance |
| HIPAA | N/A | "We don't process PHI" |

### Verifiable Claims (Isolation Demo)
```
curl -s /v1/context/inject?projects=your-project&limit=5
→ Returns ONLY your-project's observations
→ Never includes "default" scope
→ Zero observations for unknown projects
```
This is uniquely powerful — no competitor offers an API you can hit to verify isolation.

### Language Rules for Trust
| ✅ Say | ❌ Never Say |
